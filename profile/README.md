# Welcome to Precliniverse

**Precliniverse** is an open-source, modular ecosystem designed to act as a full "Operating System" for preclinical research facilities. It bridges the gap between high-level scientific rigor and digital efficiency by providing a unified, secure, and traceable environment for research data.

## 🤖 AI-Native Development Disclaimer
All code in this organization is created using advanced AI tools, under strict human supervision. We apply a strong constraint on **code quality**, **security**, and **maintainability**. This approach allows us to iterate rapidly while ensuring that the architecture remains clean, documented, and professional.

## 🏗️ The Ecosystem

Precliniverse is composed of specialized bricks that can function autonomously or as a fully governed suite:

*   **[preclinilog]**: The **Notary & Orchestrator**. It provides the central project registry (**IPxxx**), shared CRM, and global navigation hub.
*   **[precliniset]**: The **Scientific Brain (LIMS)**. Handles animal group tracking, data collection, ethical-approval traceability, and molecule inventory.
*   **[precliniquote]**: The **Financial Manager**. A smart quoting system and CRM tailored for preclinical service providers.
*   **[preclinitrain]**: The **Compliance Tracker**. Dedicated to managing regulatory training and staff skills.

These bricks will be accessible through the PrecliniLab portal
